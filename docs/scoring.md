# ShinraiCheck 判定ロジック（MVP 1.0）

## 原則

1. **媒体名だけで真偽を断定しない**
2. **政府・一次資料・大学・編集責任を持つ報道機関を高めに評価**
3. **X・YouTubeなど個人投稿可能な媒体の影響は小さくする**
4. **同じ記事の転載を「複数の独立証拠」と数えない**
5. **支持だけでなく、反証・訂正情報も取得して表示する**
6. **利用したソースと点数内訳を全表示する**

## ソース信頼度の初期値

- 日本政府・自治体（`.go.jp`）: 0.98
- 国際公的機関: 0.96
- 大学・研究機関（`.ac.jp`, `.edu`）: 0.92
- 通信社・主要報道機関: おおむね 0.82〜0.93
- 東奥日報: 0.84
- Yahoo!ニュース: 0.62（集約サイトのため、元の配信社確認が必要）
- 企業公式サイト: 0.62（当事者発表として有用だが、利害関係に注意）
- X・YouTube等: 0.14
- 未登録サイト: 0.38

値は運営者が管理画面や設定ファイルから改訂することを想定しています。

## 計算要素

- ソース信頼度
- 主張文と検索結果タイトル・概要の関連度
- 支持／反証を示す表現
- 独立ドメイン数
- 関連ソース不足
- 個人投稿媒体の割合

## 重要な限界

MVPは検索結果のタイトル・概要を主に使うため、本文の文脈、皮肉、引用、数値の細部を完全には理解できません。実用版では本文取得、一次資料抽出、日本語NLIモデル、人手での媒体評価、同一記事クラスタリングを追加してください。
